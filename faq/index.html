General

dev-codelist.net is a free, public code showcase platform. It hosts ready-to-use code snippets, templates, and full-featured bot systems — primarily for Bot Designer for Discord (BDFD). Everything is presented with syntax highlighting, copy-paste support, and clear documentation so you can drop it straight into your project.

BDFD stands for Bot Designer for Discord — a mobile and desktop app that lets you build Discord bots using a custom scripting language, without needing to write traditional code. Most of the snippets on this site are written for BDFD's scripting environment.

Yes, completely. No account, no paywall, no ads. Browse, copy, and use any code you find here at no cost.

Using the Codes

Yes. All published code snippets are intended for personal use in your own Discord bots and projects. Where individual pages note specific restrictions or credits, please respect those. A mention or credit is always appreciated but not required.

Prefix commands are triggered by a custom prefix (e.g. !eval) typed in a Discord channel. Slash commands use Discord's built-in /command interface and require the bot to have the application commands scope. Both types are labeled on each code card.

Platform Features

The Editor is a browser-based code editor built into this site. You can write, edit, and review BDFD code with syntax highlighting directly in your browser — no installation required.

Helios is a Discord bot maintained by the dev-codelist.net team. The Helios Updates page tracks its latest changes, fixes, and new features separately from the main site changelog.

Yes. The site is fully responsive and works on all modern browsers and devices. Since BDFD itself is a mobile-first app, we make sure browsing and copying codes on a phone works just as well as on desktop.
